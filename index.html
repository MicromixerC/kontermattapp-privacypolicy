2. Welche Daten werden erhoben?

KonterMatt erhebt so wenige Daten wie möglich. Im Einzelnen:

a) Anonyme Nutzer-ID (Firebase Authentication)

Beim ersten Start der App wird automatisch eine zufällige, anonyme ID erstellt (via Firebase Anonymous Authentication). Diese ID ist nicht mit deinem Namen, deiner E-Mail-Adresse oder deinem Gerät verknüpft. Sie dient ausschließlich dazu, Mehrspielersitzungen technisch zu ermöglichen.

b) Mehrspieler-Spieldaten (Firestore)

Wenn du eine Mehrspielerpartie spielst, werden folgende Daten temporär in der Cloud gespeichert:

  • Raumcode und Raumeinstellungen
  • Spielzüge und Kartendaten der laufenden Partie
  • Gewählter Anzeigename (kein Klarname erforderlich)

Diese Daten werden nach Ende der Partie automatisch gelöscht und nicht dauerhaft gespeichert.

c) Einzelspieler-Modus

Im Einzelspieler-Modus werden keinerlei Daten an externe Server übertragen. Alle Spieleinstellungen (Geschwindigkeit, Tischfarbe, Botnamen usw.) werden ausschließlich lokal auf deinem Gerät gespeichert.

d) Fehlerberichte (optional)

Wenn du manuell einen Fehlerbericht über die App einreichst, werden ein Screenshot des Spielzustands sowie eine kurze Beschreibung an Firebase Firestore übermittelt. Dies geschieht nur auf ausdrückliche Nutzeraktion hin.

3. Zweck der Datenverarbeitung
  • Technische Bereitstellung des Mehrspielermodus
  • Synchronisation von Spielzügen zwischen Spielern in Echtzeit
  • Bearbeitung freiwillig eingereichter Fehlerberichte

Es findet kein Profiling, keine Werbung und kein Tracking statt.

4. Weitergabe an Dritte

Deine Daten werden nicht an Dritte verkauft oder zu Werbezwecken weitergegeben. Die App verwendet folgende Drittanbieterdienste:

  • Firebase (Google LLC) – für anonyme Authentifizierung und Echtzeit-Datenbankfunktionen im Mehrspielermodus. Firebase unterliegt den Datenschutzbestimmungen von Google: firebase.google.com/support/privacy
5. Speicherdauer
  • Mehrspieler-Raumdaten: werden nach Spielende automatisch gelöscht
  • Lokale Einstellungen: bleiben auf deinem Gerät, bis die App deinstalliert wird
  • Fehlerberichte: werden nach Bearbeitung manuell gelöscht
6. Deine Rechte (DSGVO)

Da keine personenbezogenen Daten im klassischen Sinne erhoben werden (die anonyme Firebase-ID ist dir nicht zuordenbar), sind klassische DSGVO-Auskunftsrechte eingeschränkt anwendbar. Du kannst jedoch jederzeit:

  • Die App deinstallieren – damit werden alle lokalen Daten gelöscht
  • Uns per E-Mail kontaktieren, wenn du Fragen zu gespeicherten Daten hast
7. Datensicherheit

Die Kommunikation mit Firebase erfolgt ausschließlich über HTTPS (TLS-Verschlüsselung). Die App verwendet keine eigene Kryptografie – die Datensicherheit wird vollständig durch Apples iOS- bzw. Googles Android-Betriebssystem sowie durch Firebase gewährleistet.

8. Kinder

KonterMatt ist für alle Altersgruppen geeignet. Es werden keine Daten erhoben, die eine besondere Schutzpflicht gegenüber Minderjährigen begründen würden.

9. Änderungen dieser Erklärung

Sollte sich die Datenverarbeitung wesentlich ändern, wird diese Seite aktualisiert und das Datum am Anfang angepasst.

KonterMatt is a card game for iOS and Android. Your privacy matters to us. This Privacy Policy explains what data the app collects, how it is used, and what rights you have.

Charles Simon

2. What data is collected?

KonterMatt collects as little data as possible. Specifically:

a) Anonymous user ID (Firebase Authentication)

When the app is launched for the first time, a random anonymous ID is automatically created via Firebase Anonymous Authentication. This ID is not linked to your name, email address, or device. It is used solely to enable multiplayer sessions technically.

b) Multiplayer game data (Firestore)

When you play a multiplayer game, the following data is temporarily stored in the cloud:

  • Room code and room settings
  • Moves and card data for the ongoing game
  • Chosen display name (no real name required)

This data is automatically deleted after the game ends and is not stored permanently.

c) Single-player mode

In single-player mode, no data is transmitted to external servers. All game settings (speed, table color, bot names, etc.) are stored exclusively on your device.

d) Bug reports (optional)

If you manually submit a bug report through the app, a screenshot of the game state and a short description are sent to Firebase Firestore. This only happens upon explicit user action.

3. Purpose of data processing
  • Technical operation of the multiplayer mode
  • Real-time synchronization of game moves between players
  • Handling voluntarily submitted bug reports

There is no profiling, advertising, or tracking of any kind.

4. Sharing with third parties

Your data is never sold or shared with third parties for advertising purposes. The app uses the following third-party services:

  • Firebase (Google LLC) – for anonymous authentication and real-time database functionality in multiplayer mode. Firebase is subject to Google's privacy policy: firebase.google.com/support/privacy
5. Data retention
  • Multiplayer room data: automatically deleted after the game ends
  • Local settings: stored on your device until the app is uninstalled
  • Bug reports: manually deleted after processing
6. Your rights (GDPR)

Since no personally identifiable data is collected in the traditional sense (the anonymous Firebase ID cannot be attributed to you), classical GDPR rights have limited applicability. However, you can at any time:

  • Uninstall the app – this deletes all local data
  • Contact us by email if you have questions about stored data
7. Data security

All communication with Firebase is exclusively over HTTPS (TLS encryption). The app does not use any custom cryptography – data security is entirely handled by Apple's iOS or Google's Android operating system and by Firebase.

8. Children

KonterMatt is suitable for all age groups. No data is collected that would create special protection obligations with respect to minors.

9. Changes to this policy

If data processing changes significantly, this page will be updated and the date at the top will be adjusted accordingly.
